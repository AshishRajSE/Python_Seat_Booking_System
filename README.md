# Python Seat Booking System

## 📌 Project Overview

The **Python Seat Booking System** is a simple console-based application developed using Python. It allows users to check seat availability, book available seats, cancel booked seats, and exit the program through an interactive menu. This project demonstrates the fundamentals of Python programming and basic reservation system logic.

## ✨ Features

* View the status of all seats
* Book an available seat
* Cancel a booked seat
* Input validation for seat booking
* Simple menu-driven interface
* Easy to understand and beginner-friendly code

## 🛠️ Technologies Used

* **Language:** Python 3
* **IDE:** VS Code / PyCharm / IDLE (or any Python IDE)

## ▶️ How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open the project folder in your preferred IDE or terminal.
4. Run the program using:

```bash
python seat_booking.py
```
